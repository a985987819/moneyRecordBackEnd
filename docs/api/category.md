# 分类模块 API

基础路径：`/api/categories`

## 接口列表

### 1. 获取所有分类

获取当前用户的所有分类（按支出/收入分组）。

- **方法**: `GET`
- **路径**: `/api/categories`
- **认证**: 需要

**请求参数**: 无

**响应示例**:
```json
{
  "expense": [
    {
      "id": "1",
      "name": "餐饮",
      "icon": "🍔",
      "type": "expense",
      "color": "#FF6B6B",
      "createdAt": "2025-01-01 10:00:00",
      "updatedAt": "2025-01-01 10:00:00"
    }
  ],
  "income": [
    {
      "id": "2",
      "name": "工资",
      "icon": "💰",
      "type": "income",
      "color": "#00B894",
      "createdAt": "2025-01-01 10:00:00",
      "updatedAt": "2025-01-01 10:00:00"
    }
  ]
}
```

---

### 2. 获取支出分类

获取支出分类列表。

- **方法**: `GET`
- **路径**: `/api/categories/expense`
- **认证**: 需要

**请求参数**: 无

**响应示例**:
```json
[
  {
    "id": "1",
    "name": "餐饮",
    "icon": "🍔",
    "type": "expense",
    "color": "#FF6B6B",
    "createdAt": "2025-01-01 10:00:00",
    "updatedAt": "2025-01-01 10:00:00"
  }
]
```

---

### 3. 获取收入分类

获取收入分类列表。

- **方法**: `GET`
- **路径**: `/api/categories/income`
- **认证**: 需要

**请求参数**: 无

**响应示例**: 同支出分类

---

### 4. 创建分类

创建新的分类。

- **方法**: `POST`
- **路径**: `/api/categories`
- **认证**: 需要

**请求参数**:
```json
{
  "name": "string (必填，2-20字符)",
  "icon": "string (必填，emoji或图标名)",
  "type": "expense | income | transfer | debt | reimbursement (必填)",
  "color": "string (可选，HEX颜色值，如#FF6B6B)"
}
```

**响应示例**:
```json
{
  "id": "3",
  "name": "交通",
  "icon": "🚗",
  "type": "expense",
  "color": "#4ECDC4",
  "createdAt": "2025-04-03 16:00:00",
  "updatedAt": "2025-04-03 16:00:00"
}
```

---

### 5. 更新分类

更新指定分类。

- **方法**: `PUT`
- **路径**: `/api/categories/:id`
- **认证**: 需要

**路径参数**:
- `id` - 分类ID

**请求参数**:
```json
{
  "name": "string (可选)",
  "icon": "string (可选)",
  "color": "string (可选)"
}
```

**响应示例**: 同创建分类

**错误码**:
- 404 - 分类不存在

---

### 6. 删除分类

删除指定分类。

- **方法**: `DELETE`
- **路径**: `/api/categories/:id`
- **认证**: 需要

**路径参数**:
- `id` - 分类ID

**响应示例**:
```json
{
  "message": "删除成功"
}
```

**错误码**:
- 404 - 分类不存在

---
